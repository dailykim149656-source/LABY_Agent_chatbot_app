## 1️⃣ 동의서 필수 구성요소

### 1.1 법적 필수 고지 사항 (개인정보보호법 제15조)

동의서에 반드시 포함되어야 할 4가지 사항:

| 항목 | 내용 | 귀사 적용 |
|------|------|---------|
| **수집·이용 목적** | 왜 이 정보를 수집하는가 | 안전관리, 데이터 분석, 성능 모니터링 |
| **수집 항목** | 구체적으로 어떤 정보를 수집하는가 | 아래 섹션 2 참조 |
| **보유 및 이용기간** | 얼마나 오래 보관하는가 (강조 표시) | 항목별로 명확히 기입 필수 |
| **동의 거부권** | 거부할 수 있음 + 불이익 안내 | 서비스 미제공 불이익 명시 |

⚠️ **2025년 법개정 주요사항**:
- "~등" 같은 모호한 표현 금지
- "목적 달성 시까지" 같은 불명확한 기간 금지
- 각 항목별 **구체적인 기간** 명시 필수

## 2️⃣ 귀사 데이터 수집 항목 분류

### 2.1 필수동의 vs 선택동의 판단

#### ✅ 계약 이행에 필수 (동의 불필요) - 개인정보보호법 제15조 2호

귀사 서비스 이용계약을 위해 **반드시 필요한 정보**:

| 항목 | 구분 | 수집 이유 | 보유기간 |
|------|------|---------|---------|
| email | 필수 | 로그인 키, 계약 체결 | 서비스 이용 중 + 1년 |
| password_hash | 필수 | 계약 이행(인증) | 서비스 이용 중 |
| name | 필수 | 계약 상대방 확인 | 서비스 이용 중 + 1년 |
| affiliation, department, position | 필수 | 서비스 차등 제공/권한 관리 | 서비스 이용 중 |

💡 **핵심**: 이 정보들은 서비스 제공계약 이행에 "불가피하게 필요"하므로 **별도 동의 없이 처리 가능**. 단, "처리방침"에는 반드시 고지.

---

#### 📝 선택동의 필요 (별도 동의서 필요) - 개인정보보호법 제15조 1호

계약 이행과 무관한 추가 정보:

| 항목 | 수집 목적 | 보유기간 | 동의 필수? |
|------|---------|---------|----------|
| phone | 긴급 연락, 본인 확인 | 서비스 이용 중 + 1년 | 선택 |
| contact_email | 마케팅, 공지사항 전송 | 마케팅 수신동의까지 | 선택 |

### 2.2 센서/이벤트 기반 데이터 수집 분류

#### 1️⃣ **온도·습도 데이터**

```
수집 목적: 실험실 환경 모니터링, 품질 관리
수집 항목: 온도(℃), 습도(%), 센서ID, 타임스탬프
수집 방식: 실시간 자동 수집
보유기간: 3년 (산업규격/품질보증)
민감정보: ❌ 아님
동의 필요: ✅ 필요 (계약 이행 범위 외 추가 목적)

⚠️ 법적 고려사항:
- 개인 식별 불가(센서 데이터만) → 개인정보 아님
- BUT: 센서ID와 사용자ID를 연계하면 → 개인정보화
→ 동의 필수
```

**동의서 문구예시**:
```
■ 온도·습도 데이터 수집·이용에 동의합니다
- 수집 목적: 실험실 환경 모니터링 및 품질 관리
- 수집 항목: 온도, 습도, 수집 시간, 센서 위치
- 보유기간: 3년
- 선택사항: 미동의 시에도 기본 서비스 이용 가능하나, 
           환경 모니터링 기능 제공 불가
□ 동의함    □ 동의하지 않음
```

---

#### 2️⃣ **시약고 무게 데이터**

```
수집 목적: 재고 관리, 소비 추적, 자동 주문
수집 항목: 무게(kg), 시약명, 타임스탐프, 위치정보
수집 방식: 실시간 센서 기반
보유기간: 2년 (재고 이력 관리)
민감정보: ❌ 아님
동의 필요: ✅ 필요

⚠️ 특수 고려사항:
- 시약 종류 정보 = 조직의 연구 방향 노출 가능
- 민감정보는 아니지만, "기술 유출" 우려
- 명확한 목적 제한 필수
```

**동의서 문구예시**:
```
■ 시약 재고 데이터 수집·이용에 동의합니다
- 수집 목적: 재고 관리, 소비 추적, 자동 주문
- 수집 항목: 시약 무게, 시약명, 수집 시간, 위치
- 보유기간: 2년
- 이용 제한: 재고 관리 목적 외 제3자 제공 불가
□ 동의함    □ 동의하지 않음
```

#### 3️⃣ **음성 데이터 (STT 사용 시)**

```
수집 목적: 음성 인식을 통한 명령 처리
수집 항목: 음성 파일, 타임스탐프
저장 방식: 캐시(메모리)에만 저장, DB 저장 ❌
보유기간: 요청 처리 중에만 메모리 점유
        → 처리 완료 후 즉시 폐기

민감정보: ⚠️ 매우 민감함
- 음성은 개인을 특정할 수 있는 고유 정보
- 대화 내용 분석 시 발언자의 신원, 의도 노출 가능
- 개인정보보호법상 "민감정보" 해당

동의 필요: ✅✅ 필수 (강력한 동의 필요)
법적 근거: 정보통신망법 제48조6, 개인정보보호법 제23조

⚠️ 법적 요구사항:
1. 명시적 사전동의 필수 (묵시적 동의 불가)
2. 목적 외 이용 절대 금지
3. 제3자 제공 원칙적 금지
4. 안전조치 강화 필수
5. 정보주체의 동의 철회권 명시
```

**동의서 문구예시**:
```
■ 음성 데이터 수집·이용에 동의합니다 (필수)

- 수집 목적: STT(음성인식)를 통한 명령 처리
- 수집 항목: 음성 파일, 타임스탐프, 명령어
- 저장 위치: 메모리(캐시)에만 저장
- 보유기간: 명령 처리 중에만 메모리 점유
            → 처리 완료 후 즉시 폐기
- 보유 원칙: 
  ✓ DB에 저장하지 않음
  ✓ 개인식별정보와 분리 저장
  ✓ 백업 대상 제외

* 동의 거부 시: 음성 명령 기능 미제공
               (다른 기능은 정상 이용 가능)
* 동의 철회: 언제든지 철회 가능 (설정 메뉴)

□ 동의함    □ 동의하지 않음
```

#### 4️⃣ **영상 데이터 (사고 인식 시)**

```
수집 목적: 사고 발생 감지 및 인식
수집 항목: 영상 프레임 (전체 영상 아님)
저장 방식: 인식 결과 로그만 DB 저장
           → 원본 영상은 저장 ❌

민감정보: ⚠️⚠️ 극도로 민감함
- 영상은 개인의 얼굴, 신원, 활동 패턴 노출
- 가장 강력한 개인정보 카테고리
- 개인정보보호법상 "민감정보" + "고유식별정보"

동의 필요: ✅✅✅ 매우 강력한 동의 필수
법적 근거: 정보통신망법 제48조6, 개인정보보호법 제23조

⚠️ 법적 요구사항:
1. 명시적·개별적 사전동의 필수
2. 영상 수집 구간 명확히 고지
3. 저장 범위 최소화 (로그만)
4. 열람·수정·삭제 요청권 명시
5. 강화된 보안조치 의무
6. 영상 폐기 일정 명시
```

**동의서 문구예시**:
```
■ 영상 데이터 수집·이용에 동의합니다 (필수)

- 수집 목적: 실험실 사고 발생 실시간 감지
- 수집 범위: 
  ✓ 지정된 위험 구간 영상만 수집
  ✓ 사고 발생 시에만 처리
  ✗ 연속 녹화 금지
  
- 처리 방식:
  ✓ 영상 → AI 인식 처리
  ✓ 인식 결과 로그만 DB 저장
    (예: "14:32:15 낙상사고 감지, 위치: A-2 구역")
  ✗ 원본 영상은 저장하지 않음

- 보유기간: 인식 로그 90일

- 정보주체 권리:
  ✓ 처리된 로그 열람 가능
  ✓ 오류 수정 요청 가능
  ✓ 삭제 요청 가능 (법정 기간 내)
  ✓ 동의 철회 가능 (향후부터 적용)

* 동의 거부 시: 해당 구간 영상 수집 미실시
               (안전 기능 제한)

□ 동의함    □ 동의하지 않음
```
